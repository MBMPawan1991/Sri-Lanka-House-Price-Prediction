Overview

Sri Lanka House Price Prediction is a machine learning project that estimates residential property prices across Sri Lanka using real estate listing data. The project applies data preprocessing, feature engineering, exploratory data analysis (EDA), and regression algorithms to predict house prices based on key property characteristics.

This project demonstrates an end-to-end machine learning workflow, from data collection and cleaning to model training, evaluation, and deployment.

Features

Data preprocessing and cleaning
Exploratory Data Analysis (EDA)
Feature engineering
Outlier detection and handling
Multiple regression models
Model performance comparison
Hyperparameter tuning
Feature importance analysis
Interactive price prediction
Model deployment using Streamlit

Dataset

The dataset contains property listings from Sri Lankan real estate platforms and includes features such as:

District
City
Property Type
Land Size
Floor Area
Number of Bedrooms
Number of Bathrooms
Parking Availability
Property Age
Price (Target Variable)

Goal

The objective of this project is to build an accurate and scalable machine learning model that assists buyers, sellers, investors, and real estate professionals in estimating residential property prices across Sri Lanka.
